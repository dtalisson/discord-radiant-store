# Como Usar o Bot de Tickets

## Passo 1: Configurar a Categoria
1. Use `/painel`
2. Selecione "🎫 Tickets"
3. Selecione "⚙️ Configurar Categoria"
4. Digite o ID da categoria onde os tickets serão criados

## Passo 2: Configurar Staff (Opcional)
1. Use `/painel`
2. Selecione "🎫 Tickets"
3. Selecione "👥 Configurar Staff"
4. Digite os IDs dos cargos que podem ver tickets (separados por vírgula)

## Passo 3: Adicionar Opções de Tickets
1. Use `/painel`
2. Selecione "🎫 Tickets"
3. Selecione "📝 Gerenciar Opções"
4. Clique em "➕ Adicionar Opção"
5. Preencha:
   - Nome da Opção (ex: "Dúvida sobre Produto")
   - Descrição (ex: "Tire suas dúvidas sobre nossos produtos")
   - Emoji (opcional, ex: "🛍️")

## Passo 4: Atualizar Mensagem
1. Após adicionar opções, clique em "🔄 Atualizar Mensagem"
2. Isso atualizará a mensagem no canal de tickets com as novas opções

## Como os Usuários Usam
1. Os usuários veem a mensagem no canal configurado
2. Clicam no dropdown "Selecione uma opção para abrir um ticket..."
3. Escolhem o tipo de suporte desejado
4. Um canal privado é criado automaticamente

## Visibilidade dos Tickets
- **Apenas o usuário que abriu o ticket** pode ver e interagir
- **Cargos de staff configurados** também podem ver todos os tickets
- **Outros usuários** não conseguem ver os canais de tickets

## Botões nos Tickets
- 🔴 **Encerrar Atendimento** - Fecha o ticket após 5 segundos
- 🎫 **Gerenciar Ticket** - Mostra informações do ticket
- ℹ️ **Informações** - Explica como funciona o sistema
